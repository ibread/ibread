1. Filelist

* cdc.py: 
    Parse the verilog file, split it according to clock domain, and also extract CDC paths.
* create_cdc.py:
    Parse the verilog file, extend the netlist into two copies with 2 different clock domain.
* parse_path.py:
    Analyze CDC path generated by cdc.py. Generate the constraints to add into the dofile.
* process_pattern.py:
    Analyze pattern file generated for the receiver domain, generate constraints to add into sender domain.
